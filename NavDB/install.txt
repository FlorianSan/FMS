--- Installation du Navigation Display ---

I- Plateforme de type Linux
Les commandes exécutées sont issues de Debian/Ubuntu.

 1) Installation de PostgreSql

Installation du client
$ sudo apt-get install postgresql-client

Installation du serveur
$ sudo apt-get install postgresql

Installation d'un client d'administration
$ sudo apt-get install pgadmin3

Création des utilisateurs SQL :
$ sudo -i -u postgres 

$ psql

# CREATE USER user_nd WITH PASSWORD 'nd';
# ALTER ROLE user_nd WITH CREATEDB;
# CREATE DATABASE navdb OWNER user_nd;

Pour lister les databases
# \l

Pour fermer psql
# \q 

Pour fermer postgres
# exit

Création de la structure SQL : lancement du scrip(exécution en utilisateur courant)
$ cd /NavDB
$ chmod 777 create_db.sh
$ ./create_db.sh


Si besoin : https://doc.ubuntu-fr.org/postgresql

Supprimer une base de donnée:
DROP DATABASE navdb;
