--- Exécution du Navigation Display

1) Remplissage de la base de données
Il est nécessaire de remplir la base de données pour une première utilisation. Pour cela, ilfaut lancer la JAR Parser située dans exécutables.
# java -jar Parser.jar

En cas d'erreur de mémoire (Heap Error)

# java -jar -Xmx500m Parser.jar

(adapter la valeur 500 selon votre ordinateur).

En cas de clé dupliquée : #./create_db.sh


2) Exécution du logiciel Navigation Display
Une fois la base de données rempli, il est possible d'éxécuter le logiciel Navigation Display. Pour cela,
il faut lancer la JAR Navigation Display située dans exécutables.
# java -jar NavigationDisplay.jar
